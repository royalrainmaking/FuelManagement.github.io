  วิธีแก้ไขปัญหา Activity Log แสดง "unknown"
═══════════════════════════════════════════════════════════════════

ปัญหา:
  Activity Log แสดง "unknown" แทนที่จะแสดงข้อความที่มีความหมาย

สาเหตุ:
  Google Apps Script ยังไม่มีฟังก์ชัน getTransactionLogs หรือ
  ฟังก์ชันที่มีส่งข้อมูลกลับมาในรูปแบบที่ไม่ถูกต้อง

วิธีแก้ไข (ทำตามขั้นตอน):
───────────────────────────────────────────────────────────────────

1. เปิด Google Sheets
   https://docs.google.com/spreadsheets/d/18EaX2GwcZaPFXzcL0B9T4gFRAOhBXdHqZHm1bMJ8-sE/edit

2. ไปที่เมนู "ส่วนขยาย" > "Apps Script"

3. คัดลอกโค้ดจากไฟล์ "google-apps-script-transaction-log.js"
   และวางลงใน Apps Script Editor

4. คลิก "Deploy" > "Manage deployments"
   - คลิกไอคอน Edit (รูปดินสอ)
   - เลือก "New version"
   - คลิก "Deploy"

5. ปิด Apps Script Editor

6. เปิดหน้าเว็บระบบจัดการน้ำมัน (index.html)

7. กด F5 เพื่อ Refresh หน้าเว็บ

8. ตรวจสอบ Activity Log ว่าแสดงข้อความที่ถูกต้องหรือไม่

───────────────────────────────────────────────────────────────────

ผลลัพธ์ที่คาดหวัง:
  ✅ "ซื้อน้ำมันจาก ปตท. 10,000 ลิตร | โดย กฤษฎา ลุนาบุตร"
  ✅ "จ่ายน้ำมัน 200 ลิตร | โดย กฤษฎา ลุนาบุตร"
  
  แทนที่จะเป็น:
  ❌ "unknown 10,000 ลิตร | โดย กฤษฎา ลุนาบุตร"

───────────────────────────────────────────────────────────────────

ไฟล์ที่เกี่ยวข้อง:
  - google-apps-script-transaction-log.js (โค้ดที่ต้องเพิ่ม)
  - GOOGLE_APPS_SCRIPT_SETUP.md (คำแนะนำแบบละเอียด)
  - TRANSACTION_LOG_FIX_SUMMARY.md (สรุปการแก้ไข)
