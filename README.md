# ✈️ ระบบจัดการน้ำมันเชื้อเพลิงอากาศยาน 🛢️

## 🌟 ภาพรวมระบบ

ระบบจัดการน้ำมันเชื้อเพลิงอากาศยานสำหรับ **กรมฝนหลวงและการบินเกษตร กระทรวงเกษตรและสหกรณ์** 

### ✨ ฟีเจอร์หลัก

- 📝 **บันทึกข้อมูลการใช้น้ำมัน** - บันทึกรายละเอียดการเติมน้ำมันแต่ละครั้ง
- 📊 **Dashboard แบบ Real-time** - ติดตามข้อมูลและสถิติการใช้น้ำมัน
- 🛢️ **จัดการยอดคงเหลือ** - ติดตามยอดคงเหลือน้ำมันในแท๊งฟาร์ม, รถน้ำมัน, และถัง 200L
- 💰 **จัดการงบประมาณ** - ติดตามการใช้งบประมาณแต่ละภารกิจ
- 📤 **ส่งออกข้อมูล** - ส่งออกรายงานเป็น CSV
- 🧪 **ระบบทดสอบ UI** - ทดสอบการทำงานของระบบ

### 🎨 UI/UX ใหม่ล่าสุด

- 🌈 **สีสันสดใส**: ใช้โทนสี Cute Pink, Purple, Blue
- ✨ **Animations น่ารัก**: Floating bubbles, shimmer effects, pulse animations
- 🎯 **ปุ่มสวยงาม**: ปุ่มกลมมน มี gradient และ hover effects
- 📱 **Responsive Design**: รองรับทุกขนาดหน้าจอ
- 🚀 **Performance ดี**: โหลดเร็ว animations ลื่นไหล

## 🔧 การตั้งค่า API

ระบบใช้ Google Sheets เป็นฐานข้อมูล ตั้งค่าใน `config.js`:

```javascript
window.APP_CONFIG = {
  apiUrl: "YOUR_GOOGLE_APPS_SCRIPT_URL",
  sheetsId: "YOUR_GOOGLE_SHEETS_ID",
  sheetsUrl: "YOUR_GOOGLE_SHEETS_URL"
};
```
